# 🌤️ Taksentina Weather Web App

## Overview
**Taksentina Weather Web App** is a smart weather monitoring system developed as part of our Bachelor's degree project.  
It leverages **Internet of Things (IoT)** concepts and modern **web technologies** to collect, process, and visualize real-time meteorological data.

Our goal was to design and implement a complete solution that connects IoT-based weather sensors to a responsive web interface, enabling users to observe and analyze environmental parameters efficiently.

## 🧠 Key Features
- Real-time weather data collection and display (temperature, humidity, pressure, wind speed, etc.)
- Integration with **OpenWeather API** for live global updates  
- Dynamic and interactive **React.js** front-end  
- **Node.js** and **Express.js** back-end for API handling and real-time communication  
- Data storage and retrieval via **MongoDB**  
- Modular architecture ready for IoT device integration (sensors via WebSocket)

---

## 📊 Project Objectives
1. Collect and monitor real-time weather data from IoT sensors.  
2. Store and analyze data in a cloud database (MongoDB).  
3. Provide users with an intuitive web interface for data visualization and advice based on weather conditions.
4. Utilize RESTful APIs, such as OpenWeather API, to provide data when physical sensors are unavailable.
5. Explore the potential of IoT integration in environmental monitoring systems.
